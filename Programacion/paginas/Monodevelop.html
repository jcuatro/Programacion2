Monodevelop

Diseñado para Linux, Windows y Mac OS X soporta los lenguajes C#, Visual Basic.Net, C / C + + y Vala. Por muchos programadores no es conocido, sin embargo en su documentación (http://monodevelop.com/Documentation) nos podemos hacer una idea de todo su potencial y comenzar a utilizar este IDE para .net.
